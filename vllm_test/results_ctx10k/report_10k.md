## 结论

- 在 `1w` 输入负载下，最高完全稳定并发点是 `40`，成功吞吐约 `0.34` req/s，总 token 吞吐约 `3727.55` tok/s。

- 非 timeout error 仍然是 `0`，失败全部来自排队/推理时间过长导致的 timeout。
